Sudoku Solver

Documentation: github

Link to Project Page: Sudoku Solver

A webpage that solves any 9 X 9 Sudoku puzzle. This was a personal project I worked on this last Summer. The algorithm used is a backtracking algorithm I wrote and the program only uses a single dimensional array for encapsulating the sudoku table.
